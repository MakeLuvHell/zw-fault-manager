# 变更：添加外呼任务管理前端 UI

## 为什么
当前系统已有后端外呼任务处理能力，但缺少前端管理界面。用户需要一个可视化界面来创建、管理和触发外呼任务，包括配置数据源映射和定时执行规则。

## 变更内容
- **新增外呼任务管理页面**：包含任务列表表格、工具栏、分页功能
- **新增任务编辑弹窗**：响应式表单布局，包含四个逻辑板块
  - 基础配置：任务名称、Cron 表达式、状态开关
  - 数据源配置：跨 Schema 选择源数据表
  - 业务参数映射：固定 API 参数到数据库列名的映射配置
  - 备注信息
- **新增立即执行功能**：异步触发外呼任务，无需等待定时调度
- **新增后端 CRUD API**：支持外呼任务的增删改查及立即执行
- **Schema 感知功能**：后端元数据服务支持查询所有 Schema 和表字段
