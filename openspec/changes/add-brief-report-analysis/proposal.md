## 为什么

目前对于易问单工单数据（如月度Excel清单）缺乏高效、自动化的分析手段。人工进行多维度（如分类、处理人、时长）统计耗时耗力且难以快速产出深度业务洞察。引入基于 LLM 的智能简报功能，可实现从原始数据到结构化分析报告的自动化，帮助管理层快速掌握业务痛点、人员效能及优化方向。

## 变更内容

1.  **后端分析引擎**: 新增基于 Pandas 的 Excel 数据解析与统计模块，支持按“自分类”、“一线人员身份”等维度进行数据聚合与高频词提取。
2.  **LLM 集成**: 集成 Dify 工作流，将结构化的统计摘要数据（而非全量原始数据）传递给 LLM，生成高质量的业务分析简报。
3.  **数据持久化**: 新增 `BriefReport` 数据模型，用于存储生成的分析报告历史记录（包含源文件信息、关注点、报告内容等）。
4.  **前端交互**: 新增“智能分析报告”功能页面，支持上传 Excel、输入分析关注点、查看及导出分析报告。

## 功能 (Capabilities)

### 新增功能
- `brief-report`: 易问单智能分析简报生成与管理功能。包含数据上传预处理、多维度统计分析、LLM 报告生成及历史记录管理。
