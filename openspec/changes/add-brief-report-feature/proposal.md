## 为什么

目前用户面临海量“易问单”工单数据难以快速提炼核心价值的问题。通过人工分析 Excel 报表耗时耗力，且难以发现深层次的业务规律。现在引入 AI 智能分析功能，可以秒级生成精炼的业务简报，为决策提供数据支撑。

## 变更内容

- **新增后端模块 `module_brief`**: 采用插件化架构，负责业务逻辑处理。
- **Excel 数据解析**: 实现对工单报表（标题、时长、类型、处理人、处理时间等）的自动化解析与 JSON 转换。
- **Dify 工作流集成**: 接入“百色报表推送数据分析流”，利用大模型能力生成高质量简报。
- **持久化存储**: 新增数据库表存储原始数据快照与生成的 Markdown 报告。
- **前端分析界面**: 提供文件上传、分析进度展示及 Markdown 报告预览功能。

## 功能 (Capabilities)

### 新增功能
- `brief-report`: 提供从 Excel 上传到 AI 智能简报生成的全流程自动化能力。

### 修改功能
- 无

## 影响

- **API**: 新增 `/brief` 路由组。
- **数据库**: 新增 `wx_brief_report` 相关表（由 SQLAlchemy 模型定义）。
- **依赖**: 增加 `pandas` (Excel 处理) 和 `httpx` (异步 API 调用)。
- **前端**: 在菜单中新增“智能简报”入口，添加相关视图组件。
